Телеграм‑бот, который раз в неделю парсит страницу расписания Квиз Плиз (Москва),
ставит фильтры, нажимает «Загрузить ещё», парсит карточки игр, группирует выпуск
(название до # + номер) по всем доступным датам и публикует ОДИН опрос в чат,
если у выпуска ≥ 2 дат. В опрос всегда добавляется дополнительный вариант
«Не смогу ни в один из дней». После публикации выпуск помечается обработанным,
чтобы не дублировать опросы.

Стек: Node.js, TypeScript, grammY, node-cron, Puppeteer, Cheerio, PostgreSQL.

Команды:
- /sync — спарсить сайт и обновить БД.
- /upcoming — показать ближайшие даты (для проверки).
- /polls — отправить опросы по всем новым выпускам (≥2 дат).

Запуск:
1) Скопируйте .env.example в .env и заполните.
2) docker-compose up -d db (поднимет БД и применит sql/schema.sql).
3) npm run dev (локально) или соберите контейнер бота.
